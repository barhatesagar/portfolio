Core Generic HMIS (Health Management Information System)

Project Information

  • Category: Web Application
  • Client: Madimaze (Internal)
  • Project Date: 13 April, 2023

In the Core Generic HMIS project, I will be responsible for developing the entire application UI according to the specific requirements. The project includes the following key components:

  1. Develop a Generic Health Information Management System (HIMS) with dynamic, reusable components such as forms, tables, grids, and schedulers.
  2. Ensure seamless integration with backend services and external systems like Apache Superset.
  3. Implement responsive design and multiple theme options to enhance user experience across devices.
  4. Include key components such as dynamic forms, super grids with pagination and sorting, and schedulers for appointments.
  5. Utilize modern frameworks, emphasize customization, and conduct robust testing to ensure quality and adaptability.
  6. Create a dynamic sidebar for intuitive navigation and a dynamic toaster for real-time notifications.
  7. Integrate a generic filter and date-wise filter for efficient data management and retrieval.
  8. Focus on providing a comprehensive solution that meets the diverse needs of healthcare providers and administrators.
  9. Applied HTML, CSS, Angular, TypeScript for the Front-End.

My focus will be on creating a user-friendly, efficient, and compliant interface that meets the client's expectations and enhances the overall functionality of the HMIS.
